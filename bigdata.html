Smart City IoT Data Pipeline for Real-Time Analytics and Visualization

This project implements a scalable Smart City IoT Data Pipeline that processes, stores, and visualizes data from city sensors, such as traffic lights, air quality monitors, and energy meters. The pipeline integrates technologies like Apache NiFi for data ingestion, Apache Kafka for messaging, Hadoop HDFS for storage, Apache Spark for processing, and PostgreSQL for structured querying. Zstandard compression optimizes storage and bandwidth. Dashboards built with tools like Tableau provide real-time insights to improve city services, optimize traffic, monitor environmental conditions, and enhance public safety. Future improvements include real-time analytics and predictive modeling.

Bank Fraud Detection and Analysis with Spark

This project focuses on analyzing bank transactions to detect fraudulent activities. Data from multiple trusted sources was collected and merged into a single dataset, followed by data wrangling and filtering of unnecessary columns. Time series analysis was conducted by converting date data types, and grouping was performed on key attributes such as Account_No, Withdrawal_Date, and Transaction_Details to identify suspicious patterns. Fraudulent transactions were flagged by comparing OTPs and locations across different years. The results were stored in Elasticsearch, with Kibana used for visualizing and monitoring fraud trends.
